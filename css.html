CSS

CSS (Cascading Style Sheets) is used to style HTML content. It allows developers to control layout, colors, typography, and responsiveness. Selectors (class, id, element) target HTML elements. CSS supports Flexbox and Grid for layout, and external CSS promotes reusability and maintainability.

What I Learned

I learned to apply CSS variables, organize styles into an external file, and use Flexbox/Grid to build responsive layouts.
