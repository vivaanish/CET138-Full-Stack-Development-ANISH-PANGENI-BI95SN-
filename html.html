HTML

HTML (HyperText Markup Language) is the backbone of web pages. It structures content using semantic elements like <header>, <nav>, <main>, <article>, and <footer>. It also supports forms, tables, lists, images, and links. Semantic HTML improves SEO, accessibility, and readability.

What I Learned

I practiced writing semantic HTML, creating forms, lists, and tables, and understood how HTML provides the skeleton for a webpage.
